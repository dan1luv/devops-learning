Неделя 1 - изучаю терминал
Дата: вторник, 23 июня 2026 г. 09:48:45 (MSK)

== Урок 1 - что изучил ===
pwd, ls, cd - навигация
mkdir, touch - создание папок и файлов
echo, cat - запись и чтение 
> перезаписать >> добавить
chmod +x - сделать файл исполняемым
sed - замена текста в файле
$(команда) - подстановка результата команды
mkdir -p - идемпотентное создание папки

=== Урок 2 - что изучил ===
chmod 644/755/600/700 - числовые права
r=4 w=2 x=1 - складываешь для каждого блока
SSH ключи: приватный 600, публичный 644
ps aux - все процессы
ps aux | grep - фильтрация процессов
PID - номер процесса, по нему делаешь kill
kill / kill -9 - SIGTERM vs SIGKILL
& - запуск процесса в фоне

=== Урок 3 - что изучил ===
$PATH - список папок, где терминал ищет команды
echo $PATH | tr ':' '\n' - читаемый вид PATH
export VAR="value" - переменная окружения
$HOME - домашняя папка (то же, что ~)
alias ll="ls -la" - псевдоним команды
~/.zshrc - файл конфига zsh, выполняется при старте
source ~/.zshrc - применить изменения без перезапуска
Git - система контроля версий
